h1. What is this?

This is an example Rails application to login using facebook and vkontakte with devise and omniauth.

h2. Instructions

* Clone me <code>git clone git://github.com/mystdeim/Authproviders.git</code>
* <code>bundle install</code>
* Create and migrate the database <code>bundle exec rake db:create</code> & <code>rake db:migrate</code>
* register at "https://developers.facebook.com/apps":https://developers.facebook.com/apps and fill 'APP_ID', 'APP_SECRET' in devise.rb
* register at "http://vk.com/developers.php":http://vk.com/developers.php and fill 'APP_ID', 'APP_SECRET' in devise.rb
